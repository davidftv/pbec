# Polaris Bios Editor for Consoles (Linux/Windows, etc.)

---

## Linux/MacOS:

### Prerequisites:

gcc, etc.

### To build:

`make clean && make `

### To run:

`./pbec`

### To run tests:

`./tests`

---

## Windows:

### Prerequisites:
